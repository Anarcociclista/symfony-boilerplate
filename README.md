# Symfony boilerplate
as is


To inizialize the project run:
```
$ docker-compose up -d
```

then enter in the php container:
```
$ docker-compose exec php bash
```

and run a:
```
$ composer install
```

Your boilerplate is ready

Now add:
```
127.0.0.1 symfony.test
```
inside your /etc/hosts file and enjoy
